sc8pr 3 Documentation

sc8pr is an educational programming package for Python 3.7+ and pygame 2.0+. Inspired by Scratch and Processing, sc8pr’s aim is to make it easy for new and experienced Python programmers to create animations, games, and other graphics-based programs.

You can see some examples of sc8pr 3 by downloading the examples..

Features include:

  • a simple, event-driven approach to creating interactive animations
  • image manipulation
  • sprite management
  • physics-based collisions
  • video effects / transitions
  • a selection of GUI controls
  • curve plotting
  • robotics simulator

If your are using the earlier version, sc8pr 2, documentation is still available.
